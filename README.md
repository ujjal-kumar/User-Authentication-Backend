# User-Authentication-Backend

I'm happy to share that I've successfully implemented **JWT-based User Authentication** in my MERN Stack application.

This project focused on building a secure authentication system, allowing users to register, log in, and access protected routes.

### 🚀 Key Features

* ✅ User Registration & Login
* ✅ Password Hashing with **bcrypt**
* ✅ JWT Token Generation & Verification
* ✅ Protected Backend Routes using Authentication Middleware
* ✅ Token Storage in **localStorage**

### 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT (JSON Web Token)
* bcrypt

This task helped me gain hands-on experience with secure authentication, password protection, token-based authorization, and managing user sessions in a full-stack application.

Every project is helping me build stronger backend and full-stack development skills. Looking forward to learning and creating more!
